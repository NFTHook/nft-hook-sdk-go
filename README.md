# NFT-HOOK-SDK-GO

## Project Overview

The NFT-HOOK-SDK-GO is an integration and encapsulation package specifically designed for handling operations related to contract deployment for NFT-HOOK.

### Prerequisites

*   Go (version 1.22.3 or higher)
*   Linux environment
